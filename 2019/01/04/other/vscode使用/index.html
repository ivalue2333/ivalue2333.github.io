快捷键

  • 注释
    • 单行注释： ctrl + /
    • 多行注释：alt + shirt + a
  • 转到定义处：F12
  • 转到调用处：Alt + F12
  • 查找： ctrl + f
  • 全局查找：ctrl + shirt+ f
  • 替换: ctrl + h
  • 全局替换：ctrl + shirt + h
  • 切换到文件目录树： ctrl + shirt + e
  • 向前向后： Alt + < 或 Alt + >
  • 打开控制台：ctrl + ~
  • 查找文件：ctrl + p
  • 代码格式化: ctrl + s （golang）
  • 字体放大/缩小: ctrl + ( + 或 - )
  • 拆分编辑器 : ctrl + 1/2/3
  • 关闭编辑器窗口 : ctrl + w
  • 配置
    • 打开默认配置: ctrl + shirt + p 然后，Open Default Settings
    • 打开配置: ctrl + shirt + p 然后 open settings.json (地址：C:\Users\13722\AppData\Roaming\Code\User\settings.json)
    • 安装所有golang需要的插件
      • 打开默认配置: ctrl + shirt + p 然后，go:install，选择全部，然后确认
    • 重启vscode：ctrl + shirt + p 然后, reload window
